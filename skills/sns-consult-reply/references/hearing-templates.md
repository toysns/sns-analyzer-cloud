# ヒアリングテンプレート

## 基本原則
- 最大5問、理想は3問
- 1問1行
- 選択式にできるものは選択式に
- 「なんでも教えて」禁止
- 末尾に「わかる範囲でOK！」を付ける

---

## 分類別テンプレート

### 1. アカウント設計相談

```
ありがとう！いくつか聞かせて：
1. どのジャンルでやりたい？（料理/美容/ファッション/旅行/ビジネス/その他）
2. ターゲットは誰？（年齢層・性別・どんな人に届けたい？）
3. 顔出しあり？なし？
わかる範囲でOK！
```

### 2. 伸び悩み相談

```
ありがとう！状況把握したいから教えて：
1. アカウントURL（もしくはスクショ）見せてもらえる？
2. 今フォロワー何人くらい？
3. 週に何本くらい投稿してる？
わかる範囲でOK！
```

### 3. コンテンツ企画相談

```
ありがとう！いくつか聞かせて：
1. 今までで一番ウケた投稿ってどんなの？
2. ジャンルは何系？
3. 「こういう人に見てほしい」ってのある？
わかる範囲でOK！
```

### 4. バズ・エンゲージメント相談

```
ありがとう！確認させて：
1. 直近の投稿で一番再生された数は？
2. 平均の再生数はどれくらい？
3. コメントやいいねは投稿あたりどれくらいつく？
わかる範囲でOK！
```

### 5. 収益化相談

```
ありがとう！状況聞かせて：
1. 今フォロワー何人くらい？
2. ジャンルは何系？
3. 今やってる収益化の方法は？（案件/アフィリ/自社商品/まだなし）
わかる範囲でOK！
```

### 6. プラットフォーム固有相談

```
ありがとう！確認させて：
1. 今使ってるプラットフォームは？
2. そのプラットフォームで何が困ってる？
3. 他のプラットフォームも検討してる？
わかる範囲でOK！
```

---

## ヒアリング不要の判断基準

以下の情報が相談文に含まれていれば、ヒアリング不要で直接返信できる：

- ジャンルがわかる
- 現在の状況がわかる（フォロワー数、投稿頻度等）
- 具体的な悩みが明確

**3つ揃ってたらヒアリングスキップ → 即分析 → 即返信ドラフト**

---

## 追加ヒアリングのルール

- 1回のヒアリングで足りなかった場合、追加は **最大1回**
- 2回聞いてもまだ足りないなら、あるもので返す
- 「もっと詳しく教えて」は禁止。具体的に聞く
