# 返信パターン集

## 返信の基本構造

```
[共感/承認] 1行
[核心の指摘] 1-2行
[具体アクション] 1-3個（箇条書き）
[補足/励まし] 1行（任意）
```

**最大400文字。200文字が理想。**

---

## 分類別パターン

### 1. アカウント設計相談 → 返信パターン

**パターンA: ジャンル×フォーマット提案**
```
{ジャンル}いいね。
ぶっちゃけ{ジャンル}は競合多いから、「切り口」で差別化するのが大事。
おすすめ:
- {フォーマット辞書から該当フォーマット}で攻める
- 冒頭2秒は{具体的な冒頭設計}
- プロフィールは「{具体的なプロフ案}」
まずこの3つ決めて、10本投稿してみて。
```

**パターンB: ターゲット明確化**
```
いいね、方向性はある。
ただ「{曖昧なターゲット}」だと広すぎて刺さらんのよ。
もっと絞ろう:
- 「{具体的なペルソナ}」くらいまで絞る
- その人が「保存したくなる」情報を出す
- {具体的な投稿テーマ案}から始める
狭く深くが最短ルート。
```

## NGパターン（やってはいけない返信）

### NG1: 長すぎる
```
❌ ご相談ありがとうございます。SNSの運用についてですが、
    まず大前提として〜（以下2000文字）
```

### NG2: 抽象的すぎる
```
❌ もっと頑張りましょう！コンテンツの質を上げていけば大丈夫です！
```

### NG3: 質問攻め
```
❌ ジャンルは？ターゲットは？投稿頻度は？使ってるアプリは？
    編集時間は？参考にしてるアカウントは？収益は？...
```

### NG4: 上から目線
```
❌ それは基本ができてないですね。まずは〜を勉強してください。
```

### NG5: 逃げの回答
```
❌ 色々な要因が考えられますので、一概には言えませんが...
```
